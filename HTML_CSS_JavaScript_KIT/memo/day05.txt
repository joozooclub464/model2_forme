빈 요소, Void 요소(Empty Elements)
	여는태그와 닫는태그 사이가 없는 요소 (br)
	"단일 태그" 라고도 하며 주로 문서에 무언가를 
	첨부하기 위해 사용된다.(ex이미지 첨부)

이미지 태그
	1. 웹 페이지에서 주로 사용되는 이미지의 종류
		jpeg	: .jpg / .jpeg - 휴대폰, 카메라 사진
		gif	: .gif - 움짤, 움직이는 이미지
		png	: .png - 배경을 투명하게 할 때 사용
	2. HTML 문서에 이미지를 삽입할 때에는 <img> 태그를 사용한다.
		<img src="경로" alt="대체할문자열">
	
		상대 경로
			현재 위치한 페이지(파일) 기준으로 찾아가는 경로
			./	: 현재폴더 (ex : ./banana1.png		banana1.png)
			../	: 상위폴더
		
		절대 경로
			어떤 페이지에 있던간에 최상위 경로부터 하나씩 써놓아서
			찾아갈 수 있게 작성한 경로

하이퍼링크
	HTML 파일 링크 걸 때 사용
	현재 페이지에서 다른 페이지로 이동하고자 할 때 사용한다.
	보통 링크라고 부르며 <a> 태그를 사용한다.
	
	<a href="페이지경로" target="보여줄창">문자열 또는 이미지</a>
